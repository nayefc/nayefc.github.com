Styling Changes

After studying design for a while [1], it was easy to spot some shortcomings of this blog. I've made some styling amendments, which I will share for the benefit of anyone trying to make minor, yet impactful improvements to their website.

Font

Blogs need to be optimised for reading on the screen. There's no point on showing off nice fonts, if eyes struggle following through. I've increased the content's width by 100 pixels and increased the font size to 18px. I've also moved away from my old font style, to use a screen-friendly font. Below is my CSS diff.

-  max-width: 650px;
+  max-width: 750px;

Simplifying URLs

All posts URLs was the following format: year-month-day-title. The date is an irrelevant piece of information to be included in the URL. I've simplified all URLs to include the title only.
